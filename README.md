# hwdesk

## Installation

You can install the package via pip:

```
pip install hwdesk
```

## Usage

To start the program,

```
hwdesk
```

To specify the framerate,

```
hwdesk --fps 10
```

To exit the program, press `alt+f4` or `esc`.
